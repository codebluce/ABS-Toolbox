# ABS工具箱 Skill · 功能演示介绍

> 一个面向 ABS（资产支持证券）发行业务的 AI Agent Skill：自然语言触发，从簿记明细录入到台账加工，再到六大分析看板一键生成，全流程本地自动化。

## 一、主要功能

**簿记录入（明细 → 台账）**：业务同事只需把各项目的簿记明细 Excel 丢进指定目录，对 Agent 说一句"补充簿记数据"，`increment_merge.py` 即自动完成明细与主台账的匹配合并——内置机构名归一化（去连字符、核心名提取、硬映射三级匹配）、簿记年份纠错、17 项 QC 校验，支持"补充簿记"与"增量合并"两种模式，默认幂等可重跑。

**台账分析（一句话出六大看板）**：说"跑一个 ABS 综合看板"，主程序读取定稿台账，一次生成单文件 HTML 综合看板，含四大模块：

- **发行定价**：定价测试（选品种+利率上限 → 投资动态洞察+逐笔定价明细）、成本分布、利差分析、投资明细四个子看板；

  ![发行定价 · 定价测试](img/01_pricing.png)

- **机构统计**：管理人/销售机构/托管行三维排名，自动做券商过滤与集团实体合并；

  ![机构统计看板](img/02_institution.png)

- **投资人分析**：理财子投资矩阵与画像（Plotly 交互+月份滑块）、非标额度与授信总额度监控（摊还到期+新增认购 → 实时剩余额度）；

  ![投资人分析 · 理财子矩阵](img/03_investor.png)

- **投资台账**：1800+ 条投资记录的多维筛选统计——资产/分层/评级/机构/时间/份额区间任意组合，分组统计、透视表、明细清单三视图，一键导出 CSV；

  ![投资台账 · 多维筛选统计](img/04_ledger.png)

- **智能问答**：右下角悬浮球，用大白话问"招银理财白条投资规模"，本地语义解析自动识别机构、资产子串、业务同义词（夹层→中间级），亮出理解的条件、给出精确数字，并联动下方面板筛选器，支持追问。

  ![智能问答 · 自然语言查数](img/05_chat.png)

## 二、实现机制

纯本地 Python（pandas + openpyxl）数据管线：各模块统一 `compute_data`（含 QC 预检）+ `render_body`（HTML 渲染）双接口，主程序编排拼装为**零依赖单文件 HTML**——数据以 JSON 内嵌，前端 vanilla JS 实现筛选、透视与问答，离线可用、数据不出内网。智能问答为确定性本地语义解析，非 LLM 调用，结果可复现可审计。

## 三、开发迭代历程

由三个独立 skill（簿记录入/机构统计/发行定价）经六轮版本（v2.0→v2.5）整合而成：每轮迁移均通过"5 层自检"（字节对比/端到端穿行/逐 cell diff/原 skill 冒烟/回归测试）保证功能等价，并走 A→B→C 三角色审计循环（实现→独立审计→归档）留痕。此后经"lab 实验转正"机制孵化出投资人分析、投资台账与智能问答模块，问答模块上线前完成 30+ 用例穿行审计、修复 8 项缺陷。全程由 AI Agent 多会话协作开发，Gitee+GitHub 双仓同步，并沉淀了多 Agent 协作的版本管理规范。
