# dashboards/ 目录说明

ABS发行定价与机构统计看板的归档目录（v2.0.0 英文化）。

## 目录结构

```
dashboards/
├── 01_latest/         ← 当前最新日期的看板（快速访问入口）
├── 02_history/        ← 按日期子文件夹归档的历史版本
├── 03_test/           ← 测试/调试产物
├── 04_reference/      ← md文档类参考资料
└── README.md          ← 本文件
```

## 文件命名规则

正式看板统一命名格式：`YYYYMMDD_<看板名>.html`

| 看板名 | 来源工具 | 说明 |
|--------|---------|------|
| `发行定价分析工具` | gen_abs_cost_report.py | 成本分布分析（月份/资产类型/期限） |
| `机构投标利率看板` | gen_compare_tool.py | 机构成本交互式比对 |
| `机构投标基准利差看板` | gen_spread_report.py | 基准利差分析 |
| `机构统计看板` | gen_institution_stats.py | 机构统计三表 |

示例：`20260617_发行定价分析工具.html`

## 更新流程

每次生成新看板时：

1. **归档旧最新版本**：将 `01-最新版本/` 中的4份HTML移到 `02-历史版本/<原日期>/` 子文件夹
2. **放入新看板**：新生成的4份HTML放入 `01-最新版本/`
3. **测试产物归档**：调试或测试产物放 `03-测试文件/`，不进入正式版本

## 历史版本子文件夹命名

按日期命名：`YYYYMMDD`（如 `20260603/`）

每个历史子文件夹内保留该日期的4份完整看板（如果某次只有部分看板更新，仍保留当时所有生成的版本）。
